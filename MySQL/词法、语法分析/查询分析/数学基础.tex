\begin{definition}{(subgraph)}
    设 H = (V, E) 是一个超图，$V' \subseteq V$， $V'$ 是普通节点的一个子集。

    \ \ \ $G|_{V'}$ 定义为 $G|_{V'} = (V',E')$，其中 $E' = \{(u,v)|(u,v) \in E,u \subseteq V',v \subseteq V'\}$。
    
    \ \ \ $V'$ 上的节点排序继承 $V$ 上节点排序。
