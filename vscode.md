# VSCode如何返回上一步
在Windows中可以使用快捷键“Alt+←”实现  
在Linux中可以使用快捷键“Ctrl+Alt+ -”实现  
在Mac中可以使用快捷键“Ctrl + -”实现  

# 大括号的折叠
 Mac:Cmd + Option + 右方括号  
 Windows:Ctrl + Shift + 右方括号
